# QuestWise Learning Game MVP

QuestWise is a lightweight browser-based learning game MVP where players answer short questions, build streaks, protect hearts, and complete themed learning realms.

## Features

- Three learning realms: math, science, and vocabulary.
- Multiple-choice quiz loop with instant feedback and learning facts.
- Score, streak, heart, rank, and progress tracking.
- Responsive, playful UI for desktop and mobile.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
